# 🔐 EmojiCipher 

EmojiCipher Pro is a GUI-based encrypted emoji messaging tool built using Python.

## 🚀 Features

- Convert text messages into encrypted emoji sequences
- Resolve emoji messages back to original text
- Password-based symmetric encryption
- Unique emoji pattern generated for every message
- Modern dark-themed GUI
- Added +10 more emojis soon add all emojis 

## 🛠 Requirements

- Python 3.8+
- cryptography library

## 📦 Installation

git clone https://github.com/suraj2830009-arch/EmojiCipher.git

cd EmojiCipher-Pro

python3 -m venv venv

source venv/bin/activate   # Linux / Kali

pip install -r requirements.txt

## ▶ Run

python emoji_cipher.py

## 🔐 How It Works

1. Enter message
2. Enter secret password
3. Click Convert
4. Share emoji output
5. Receiver uses same password to resolve

Wrong password = Decryption fails.
